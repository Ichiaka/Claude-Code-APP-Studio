# /i18n-setup

## Protocolo

1. Pregunta idiomas a soportar inicialmente.
2. Elige librería según stack.
3. Define estructura de archivos de traducción.
4. Configura detección de idioma (navegador, dispositivo, manual).
5. Documenta convenciones (cómo nombrar keys, cómo extraer).

## Output

`docs/features/i18n.md` con convenciones del proyecto.
