# Código de conducta

Este proyecto adopta una versión adaptada del
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Lo que esperamos

- Trato respetuoso, profesional y constructivo en issues, discussions y PRs.
- Críticas a las ideas, no a las personas.
- Asumir buena fe en las contribuciones ajenas.
- Reconocer y agradecer el trabajo de otros.

## Lo que no toleramos

- Acoso, insultos, ataques personales.
- Comentarios despectivos por género, orientación, raza, origen, religión,
  capacidad, edad, etc.
- Doxxing o compartir información privada de otra persona sin consentimiento.
- Spam o promoción no relacionada.

## Alcance

Este código aplica a todos los espacios del proyecto: repositorio (issues,
discussions, PRs, code review, commits), y a cualquier canal oficial que se
añada en el futuro.

## Aplicación

Los mantenedores son responsables de aclarar y hacer cumplir estos estándares.
Pueden editar, borrar o rechazar contribuciones que violen este código, y pueden
banear temporal o permanentemente a quienes lo violen repetidamente.

## Cómo reportar

Si presencias o sufres una violación de este código, abre un issue privado
contactando con el mantenedor del repo (vía el email visible en el perfil del
mantenedor o GitHub), o usa el mecanismo de "Report content" de GitHub.

Todos los reportes se revisan con confidencialidad y se actúa de buena fe.
